Dashboard Panel Queries (Configured in Grafana)
-----------------------------------------
Daily Incidents (Time Series panel)
Query:
SELECT
  date AS time,
  COUNT(incident_id) AS incident_count
FROM incidents
GROUP BY date
ORDER BY date;

Top Event Types (Pie Chart panel)
Query:
SELECT
  event_type,
  COUNT(*) AS event_count
FROM events
GROUP BY event_type
ORDER BY event_count DESC;
