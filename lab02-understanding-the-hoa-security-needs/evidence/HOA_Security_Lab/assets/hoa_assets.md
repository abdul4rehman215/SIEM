# HOA Assets Inventory

## Physical Assets
- CCTV cameras (common areas, entrances, parking)
- DVR/NVR recording device (on-site or cloud-managed)
- Access control systems (gate/keycard systems)
- Office computers/laptops used by HOA staff/board
- Printers/scanners used for resident documentation
- Network equipment (router, switches, Wi-Fi access points)
- Storage devices (external drives/USBs used for backups)

## Digital Assets
- Resident database (names, addresses, contact details)
- HOA email accounts (board@, admin@, support@)
- Financial systems (budget spreadsheets, banking portals, invoices)
- CCTV footage storage (NAS/cloud folders/DVR storage)
- HOA website/portal (announcements, payment portal, documents)
- Meeting documents (minutes, notices, voting records)
- Vendor contracts and maintenance records

## Third-Party / External Dependencies
- ISP modem/router and service provider account
- Cloud storage (Google Drive/Dropbox/OneDrive, etc.)
- Payment processors / online banking services
- Vendor remote access (CCTV installer, IT support)
