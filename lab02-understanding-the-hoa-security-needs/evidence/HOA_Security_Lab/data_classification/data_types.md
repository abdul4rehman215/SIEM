# HOA Data Classification

## High Sensitivity (Confidential)
- Personally Identifiable Information (PII)
  - Resident names, addresses, phone numbers, emails
  - Emergency contacts
  - Access credentials (portal passwords, Wi-Fi passwords if linked)
- Financial Records
  - Bank account details, payment history
  - Budget and expenditure records
  - Vendor invoices and contracts with pricing

## Medium Sensitivity (Internal Use)
- Operational Data
  - CCTV logs and video footage (may contain identifiable individuals)
  - Maintenance schedules
  - Visitor logs / gate access logs
  - HOA internal emails and communications

## Low Sensitivity (Public/Shared)
- Public-facing HOA notices
- Meeting summaries intended for residents
- General HOA rules/regulations posted on the portal/notice board

## Notes
- CCTV footage is operational data but can become confidential if it is used in an incident investigation.
- Data classification should guide access control, retention, and encryption requirements.
